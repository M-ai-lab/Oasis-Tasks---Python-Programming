# Oasis-Tasks---Python-Programming
**Calculator**

Description:
A command-line calculator application developed in Python to perform basic arithmetic operations including addition, subtraction, multiplication, and division. The project demonstrates core programming concepts such as user input handling, conditional logic, function-based design, and basic error handling for invalid inputs.

Tech Stack: Python
Key Concepts: Control flow, functions, input validation

**Random Password Generator**

Description:
A Python utility that generates secure, random passwords based on user-defined criteria such as length and character sets (uppercase, lowercase, digits, and special characters). This project focuses on randomness, string manipulation, and building small security-focused tools for practical use.

Tech Stack: Python
Key Concepts: Randomization, string operations, user configuration

**Basic Weather App**

Description:
A Python-based weather application that retrieves and displays real-time weather information for a given city using a public API. The project demonstrates making HTTP requests, parsing JSON responses, handling API keys securely, and presenting external data in a user-friendly format.

Tech Stack: Python, REST API
Key Concepts: API integration, JSON parsing, error handling

**Voice Assistant**

Description:
A voice-controlled assistant built with Python that uses speech recognition and text-to-speech to interact with users. The assistant processes voice commands and performs basic actions such as responding to queries, providing time information, or opening web resources. This project highlights integration of third-party libraries and building interactive applications.

Tech Stack: Python, Speech Recognition, Text-to-Speech
Key Concepts: Audio I/O, library integration, event-driven logic

**To-Do List / File Handling App**

Description:
A simple file-based task management application built with Python that allows users to create, view, update, and delete tasks. The project demonstrates file handling for data persistence and implements basic CRUD operations, emphasizing practical application development with persistent storage.

Tech Stack: Python
Key Concepts: File I/O, CRUD operations, data persistence
